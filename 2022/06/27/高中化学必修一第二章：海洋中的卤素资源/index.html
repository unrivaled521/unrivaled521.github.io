前情提要

不同区域化学教材可能不同，该笔记基于上海科学技术出版社普通高中教科书化学必修第一册（上海地区适用）

高中化学必修一第二章：海洋中的卤素资源

2.1 海水中的氯

一、氯气的性质

1.黄绿色刺激性气味的有毒气体，密度大于空气

2.能溶于水(1：2)，其水溶液称为氯水

3.易液化形成液氯

二、氯气的化学性质

1.氯气和金属反应

(1)：氯气和铁反应：

$2Fe$ + $3Cl_2$ $\stackrel{点燃}{==}$ $2FeCl_3$

现象：褐色的烟

(2)：氯气和钠反应：

$2Na$ + $Cl_2$ $\stackrel{点燃}{==}$ $2NaCl$

现象：发出黄光，产生白色固体

(3)：氯气和铜反应：

$Cu$ + $Cl_2$ $\stackrel{点燃}{==}$ $CuCl_2$

现象：棕黄色的烟

氯气与变价金属反应均生成该金属的最高价的氯化物

2.氯气与非金属反应

氯气与氢气反应：

$H_2$ + $Cl_2$ $\stackrel{点燃}{==}$ $2HCl$ (工业制盐酸)

现象：安静的燃烧，发出苍白色火焰，瓶口有白雾产生

3.氯气与水反应

$Cl_2$ + $H_2O$ $\rightleftharpoons$ $HCl$ + $HClO$

可逆反应：同一条件下，同时向正(右)、逆(左)两个方向进行的化学反应

4.氯气与碱反应

(1)：氯气与氢氧化钠反应：

$2NaOH$ + $Cl_2$ = $NaCl$ + $NaClO$ + $H_2O$

用途：实验室中常用氢氧化钠溶液来吸收氯气的尾气，制消毒剂，制漂白液

(2):氯气与氢氧化钙反应：

$2Ca(OH)_2$ + $2Cl_2$ = $CaCl_2$ + $Ca(ClO)_2$ + $2H_2O$

用途：制漂粉精

漂粉精：

主要成分：氯化钙，次氯酸钙

有效成分：次氯酸钙

三、次氯酸

1.弱酸性：次氯酸是一元弱酸，比碳酸弱

2.漂白性：由于次氯酸具有强氧化性，可永久漂白有机色质，可杀菌消毒

3.不稳定性：$2HClO$ $\stackrel{光照}{==}$ $2HCl$ + $O_2$ $\uparrow$

新制氯水 久制氯水（盐酸）
物质 $H_2O$，$Cl_2$，$HCl$，$HClO$ $H_2O$，$HCl$
微粒 分子：$H_2O$，$Cl_2$，$HClO$ 分子：$H_2O$，$HCl$
离子：$H^+$，$Cl^-$，$ClO^-$，$OH^-$ 离子：$H^+$，$Cl^-$，$OH^-$
性质 黄绿色，酸性，漂白性，强氧化性 酸性
干燥氯气 湿润氯气
有色布条 无明显现象 褪色($HClO$)
蓝色石蕊试纸 无明显现象 先变红后边白($HCl$，$HClO$)
淀粉碘化钾试纸 无明显现象 变蓝($Cl$ + $2KI$ = $I_2$ + $2KCl$)

碘单质遇淀粉变蓝：$I_2$ + $5Cl_2$ + $6H_2O$ = $10HCl$ + $2HIO_3)$

四、漂粉精的漂白原理

$Ca(ClO)_2$ + $CO_2$ + $H_2O$ = $CaCO_3$ + $2HClO$

$Ca(ClO)_2$ + $HCl(稀)$ = $CaCl_2$ + $2HClO$

五、粗盐提纯及氯碱工业

1.海水晒盐

2.粗盐的提纯

(1)：步骤：称量粗盐 $\rightarrow$ 溶解 $\rightarrow$ 过滤 $\rightarrow$ 蒸发 $\rightarrow$ 洗涤 $\rightarrow$ 烘干 $\rightarrow$ 称量精盐

(2)：注意要点

①溶解时，水太少，食盐不能溶解，水太多蒸发时间太长

②过滤时，一贴、二低、三靠

③蒸发时，待出现多量固体时，停止加热，否则固体飞溅

④少量水洗涤时，可溶性杂质量少被除去，食盐量多只损失很少量

⑤4次用到玻璃棒，分别起加速溶解、引流、使受热均匀、转移晶体的作用

3.除去 $Ca^{2+}$，$Mg^{2+}$，$SO_4^{2-}$

(1)：除杂原则：不引入新杂质或引入的新杂质需通过后面的试剂除去，除杂试剂适当过量，成本低

(2)：选用什么试剂？

$Ca^{2+}$ —— $Na_2CO_3$

$Mg^{2+}$ —— $NaOH$

$SO_4^{2-}$ —— $BaCl_2$

(3)：加入 $Na_2CO_3$ ， $NaOH$ ， $BaCl_2$ 的顺序：先加 $BaCl_2$ 再加 $Na_2CO_3$

(4)：如何除去由于加入过量的 $Na_2CO_3$ , $NaOH$ 所引入的过量 $CO_3^{2-}$ 和$OH^-$呢？

答：加入盐酸

(5)：加入盐酸在过滤前，还是在过滤后进行？为什么？

答：过滤后加入盐酸。否则沉淀溶解

4.中国现代化工之母——氯碱工业

电解饱和食盐水

阴极：与电源负极相连的电极

阳极：与电源正极相连的电极

现象：阴阳两极产生气泡，阴极附近溶液变红

产物检验：

①阳极气体能使湿润的淀粉碘化钾试纸变蓝($Cl_2$)

②将阴极产生的气体收集，可进行爆鸣实验($H_2$)

③阴极附近溶液能使酚酞变红($NaOH$)

5.电离

(1).定义：电解质（如酸、碱、盐）在水溶液中或受热融化时产生自由移动离子的过程
(2).电离方程式——遵循电荷守恒

6.烧碱($NaOH$)

(1)：俗称：烧碱，火碱，苛性钠

(2)：物理性质：

①固体，易潮解，可用做干燥剂

可干燥中性气体($H_2$ , $O_2$)，碱性气体($NH_3$)

②溶于水放出大量的热

(3)：化学性质：一元强碱，具有碱的通性，与酸、某些盐、酸性氧化物反应，使酸碱指示剂变色

(4)：用途：炼铝，纺织，制皂，精炼石油，印染，造纸

7.盐酸(氯化氢的水溶液)

(1)：盐酸

①物理性质：无色有刺激性气味的溶液，易挥发，浓盐酸瓶口有白雾

②化学性质：一元强酸，具有酸的通性，与碱、某些盐、碱性氧化物、活泼金属反应，
使酸碱指示剂变色

③用途：生活和工业中被广泛应用

(2)：氯化氢

①物理性质：无色有刺激性气味的气体，密度大于空气，极易溶于水（1:500）

注：

a.干燥氯化氢气体不能使干燥蓝色石蕊试纸变红

b.干燥氯化氢气体不能电离，无导电性

②喷泉实验：

现象：水被倒吸上来，形成喷泉，紫色石蕊变红

结论：氯化氢气体极易溶于水

原理：胶头滴管内的水挤入烧瓶，溶解大量的HCl气体，使烧瓶内气压远小于外界大气压，烧杯内的水被压入烧瓶

本质：瓶内压强 < 瓶外压强

注：利用此装置，还能形成喷泉的组合有？

答：①$NH_3$ 、$H_2O$ ②$CO_2$ 、$NaOH$ ③$HCl$ 、$NaOH$

③工业制法

原理：$H_2$ (过量) + $Cl_2$ $\stackrel{点燃}{==}$ $2HCl$

装置：内管通氯气，外管通氢气

目的是过量的氢气包裹氯气，使有毒的氯气充分反应

④实验室制氯化氢

原理：$NaCl$ + $H_2SO_4(浓)$ $\stackrel{微热}{==}$ $NaHSO_4$ + $HCl$ $\uparrow$

思考：

a.石蕊试液喷入烧瓶后，为什么会变红？——有盐酸生成

b.烧杯中的溶液为什么会喷入烧瓶？——烧瓶内压强减小

c.为什么烧瓶内外的压强相差很大，氯化氢哪里去了？——氯化氢溶解在水里

d.为什么烧瓶内最终仍有少量气体？—— 氯化氢中混有空气

e.什么情况下烧瓶中会充满液体？——氯化氢很纯净

f.6.怎样才能使喷泉实验做得成功？——烧瓶要干燥，气密性好， 氯化氢气体充满烧瓶

2.2 氧化还原反应和离子反应

一、氧化还原反应

口诀：

氧化剂：降得还

还原剂：升失氧

得失氧(表象) $\rightarrow$ 化合价升降(特征，判断依据) $\rightarrow$ 电子转移(得失和偏移)(本质)

二、单线桥法标出电子转移方向和数目

箭头方向：反应物中，升失元素指向降得元素

数目：转移电子数目 = 升价总数 = 降价总数 = 失电子总数 = 得电子总数

三、四种基本反应和氧化还原的关系

四、氧化性和还原性

1.氧化性：氧化剂具有氧化性，得电子能力强，化合价降低

2.还原性：还原剂具有还原性，失电子能力强，化合价升高

五、规律

1.强制弱规律

氧化剂(强氧化剂) + 还原剂(强还原剂) = 还原产物(弱还原剂) + 氧化产物(弱氧化剂)

2.价态规律

元素最高价态时只具有氧化性

元素最低价态时只具有还原性

元素处于中间价态时既有氧化性又有还原性

六、常见的氧化剂和还原剂

七、离子反应(电解质与非电解质)

1.物质导电：存在自由带电的微粒

①自由电子：金属单质，石墨

②自由离子：电解质的水溶液或熔融状态

2.化合物的分类

(1)：电解质：在水溶液中或熔化状态下本身能够导电的化合物(酸，碱，盐，部分金属，水)

(2)：非电解质：在水溶液中或熔化状态下本身都不能够导电的化合物(大部分的有机物，部分非金属： $CO_2$，$SO_2$，$SO_3$，$NH_3$)

①固体：金属单质，石墨

②在水溶液和熔化状态：盐(除 $AlCl_3$)，强碱

③只在水溶液导电：酸，弱碱，$AlCl_3$

3.电解质的电离

(1)：电解质在水溶液或熔化状态时，产生自由离子的过程

(2)：电离方程式

完全电离——强电解质 $HCl$ = $H^+$ + $Cl^-$

部分电离——弱电解质 $CH_3COOH$ $\rightleftharpoons$ $CH_3COO^-$ + $H^+$

4.离子反应

(1)：定义：有离子参加的反应

(2)：实质：向离子浓度减小的方向进行

(3)：特点：一般在水溶液中进行，如复分解、置换等

5.离子方程式

(1)：写：写出化学方程式
(2)：拆：把易溶与水的强电解质（强酸、强碱、可溶性盐）拆写成离子形式
(3)：删：删去方程式两边不参加反应的离子
(4)：查：查原子个数和电荷数前后是否相等

6.离子不共存——会发生反应

(1)：复分解：生成沉淀，气体，水，弱酸，弱碱

与 $H^+$ 不能共存：$OH^-$，$CH_3COO^-$，$F^-$，$ClO^-$，$S^{2-}$，$CO_3^{2-}$，$SO_3^{2-}$，$PO_4^{3-}$，$HS^-$，$HCO_3^-$，$HSO_3^-$，$HPO_4^{2-}$，$H_2PO_4^-$

与 $OH^-$ 不能共存：$H^+$，$NH_4^+$，$Mg^{2+}$，$Al^{3+}$，$Fe^{3+}$，$Fe^{2+}$，$Cu^{2+}$，$Zn^{2+}$，$Ag^+$，$HS^-$，$HCO_3^-$，$HSO_3^-$，$HPO_4^{2-}$，$H_2PO_4^-$

不能共存($Cl^-$，$Ag^+$) $\Longrightarrow$ 碳酸银沉淀，氯化银沉淀，溴化银沉淀，碘化银沉淀

$CO_3^{2-}$，$SO_3^{2-}$，$S^{2-}$ 三种离子除与 $K^+$，$Na^+$，$NH_4^-$ 共存外，与其他阳离子均不共存

有色离子：$Cu^{2+}$ 蓝色，$Fe^{3+}$ 黄色，$Fe^{2+}$ 浅绿色，$MnO_4^-$ 紫红色

(2)：氧化还原反应：
与 $Fe^{3+}$ 不能共存：$I^-$，$S^{2-}$，$HS^-$

与 $H^+$ ，$NO_3^-$ 不能共存：$Br^-$，$I^-$，$S^{2-}$，$SO_3^{2-}$，$Fe^{2+}$，$HS^-$，$HSO_3^{-}$

与 $MnO_4^-$ ($H^+$) 不能共存：$Cl^-$，$Br^-$，$I^-$，$S^{2-}$，$SO_3^{2-}$，$Fe^{2+}$，$HS^-$，$HSO_3^{-}$

2.3 溴和碘的提取

一、卤素原子结构

相同点：最外层电子数均为7，易得1 e,化学性质相似

不同点：电子层数逐步增加，性质具有递变性

二、卤素单质的物理性质

液溴($Br_2$)：易挥发，水封保存于棕色瓶,玻璃塞

固体碘单质($I_2$)：遇淀粉变蓝，易升华

三、卤素单质的化学性质

1.置换反应

活泼性：$F_2$ > $Cl_2$ > $Br_2$ > $I_2$

$Cl_2$ + $2NaBr$ = $Br_2$ + $2NaCl$

$Cl_2$ + $2KI$ = $I_2$ + $2KCl$

$Br_2$ + $2KI$ = $I_2$ + $2KBr$

2.与氢气

$H_2$ + $F_2$ $\stackrel{黑暗}{==}$ $2HF$

$H_2$ + $Cl_2$ $\stackrel{点燃}{==}$ $2HCl$

$H_2$ + $Br_2$ $\stackrel{500°C}{==}$ $2HBr$

$H_2$ + $I_2$ $\stackrel{\Delta}{\rightleftharpoons}$ $2HI$

3.与水

$2F_2$ + $H_2O$ = $4HF$ + $O_2$ $\uparrow$ (没有+1价的氟)

$Cl_2$ + $H_2O$ $\rightleftharpoons$ $HCl$ + $HClO$ (能反应)

$Br_2$ + $H_2O$ $\rightleftharpoons$ $HBr$ + $HBrO$ (很弱)

$I_2$ + $H_2O$ $\rightleftharpoons$ $HI$ + $HIO$ (极弱)

结论：活泼性：$F_2$ > $Cl_2$ > $Br_2$ > $I_2$

四、卤素的用途

五、怎样检验 $Cl^-$，$Br^-$，$I^-$

1.可检验 $Cl^-$，$Br^-$，$I^-$ 的存在

$AgNO_3$ + $NaCl$ = $NaNO_3$ + $AgCl$ $\downarrow$ (白色)

$AgNO_3$ + $NaBr$ = $NaNO_3$ + $AgBr$ $\downarrow$ (淡黄色)

$AgNO_3$ + $KI$ = $KNO_3$ + $AgI$ $\downarrow$ (黄色)

$Na_2CO_3$ + $2AgNO_3$ = $2NaNO_3$ + $Ag_2CO_3$ $\downarrow$ (白色)

检验 $Cl^-$，为排除 $CO_3^{2-}$ 先加稀 $HNO_3$ ,再加 $AgNO_3$ 溶液

2.可检验 $Br^-$，$I^-$ 的存在

3.$I^-$ 的检验：在溶液样品中加入氯水，再加淀粉溶液，如果变蓝，说明含有 $I^-$

4.卤化银的性质（氟化银除外）：

(1)：不溶于水，也不溶于酸和碱

$AgCl$ $\rightarrow$ 白色沉淀

$AgBr$ $\rightarrow$ 淡黄色沉淀

$AgI$ $\rightarrow$ 黄色沉淀

(2)：不稳定性：$2AgX$ $\stackrel{光照}{==}$ $2Ag$ ＋ $X_2$

5.用途：

(1)：指纹鉴定：法医做指纹显影试验时，先对准指纹处喷洒 $AgNO_3$ 溶液，使之与人体分泌出的汗水中的某种离子结合，然后进行光照处理，就可显出指纹

原理：人体汗液中含有 $NaCl$ ，加入的 $AgNO_3$ 与之结合成 $AgCl$，$AgCl$ 见光分解生成黑色的银，从而显出指纹

$AgNO_3$ + $NaCl$ = $AgCl$ $\downarrow$ + $NaNO_3$

$2AgCl$ = $2Ag$ + $Cl_2$

(2)：感光纸、胶卷、变色眼镜($AgBr$)

原理：$2AgBr$ $\stackrel{光照}{\rightleftharpoons}$ $2Ag$ + $Br_2$

(3)：人工降雨($AgI$)

大家知道，有雨必先有云，但是有云不一定有雨。自然界过冷云降雨(或雪)是由于云中除小水滴外，还有足够的冰晶──饱和水汽或过冷却水滴在冰核(不溶于水的尘粒)作用下凝华或冻结而形成的冰相胚胎。过冷云中水滴的水分子会不断蒸发并凝华到冰晶上，冰晶不断长大以致下落为雪，如果云下气温高于0 ℃，它们就会融化成雨。碘化银在人工降雨中所起的作用在气象学上称作冷云催化。碘化银只要受热后就会在空气中形成极多极细的碘化银粒子。1 g碘化银可以形成几十万亿个微粒。这些微粒会随气流运动进入云中，在冷云中产生几万亿到上百亿个冰晶。
