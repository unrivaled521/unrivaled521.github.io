高中化学必修一第一章：化学研究的天地

前情提要

不同区域化学教材可能不同，该笔记基于上海科学技术出版社普通高中教科书化学必修第一册（上海地区适用）

高中化学必修一第一章：化学研究的天地

1.1 物质的分类

一、物质分类

1.

常见的酸：

七大强酸：盐酸($HCl$),硝酸($HNO_3$)，硫酸($H_2SO_4$)，溴酸($HBr$)，碘酸($HI$)，氯酸($HClO_3$)，高氯酸($HClO_4$)

弱酸：醋酸($CH_3COOH$)，次氯酸($HClO$)，，碳酸($H_2CO_3$)

常见的碱：

四大强碱：氢氧化钠($NaOH$)，氢氧化钾($KOH$)，氢氧化钙($Ca（OH）_2$），氢氧化钡($Ba（OH）_2$）

弱碱：氢氧化铝($Al(OH)_3$)，氢氧化镁($Mg(OH)_2$)，一水合氨($NH_3*H_2O$)

常见的盐：

可溶性盐：钾，钠，铵，硝盐均可溶

不可溶性盐：氯化银($Agcl$)，溴化银($AgBr$)，碘化银($AgI$)，碳酸银($Ag_2CO_3$)，硫酸钡($BaSO_4$)，碳酸钡($BaCO_3$)，亚硫酸钡($BaSO3$)




类别$\rightarrow$结构$\rightarrow$性质$\rightarrow$用途

例：

1.二氧化碳通入澄清石灰水，先变浑浊后又澄清

$CO_2$ + $Ca(OH)_2$ = $CaCO_3$ $\downarrow$ + $H_2O$

$CaCO_3$ + $CO_2$ + $H_2O$ = $Ca(HCO)_2$

2.二氧化硫通入澄清石灰水，先变浑浊后又澄清

$SO_2$ + $Ca(OH)_2$ = $CaSO_3$$\downarrow$ + $H_2O$

$CaSO_3$ + $SO_2$ + $H_2O$ = $Ca(HSO)_2$

2.


物质根据聚集状态分为固液气三态

(1):

物质状态 是否有固定形状 是否有固定体积 微粒间距 微粒间的作用力 压缩
气体 否 否 极大 极弱 极易
液体 否 是 小 弱 弱
固体 是 是 极小 强 极弱

(2):物质的三态变化

①本质：微粒间距和作用力改变

②措施：改变温度、压强

固体（升温）$\rightarrow$液体（升温）$\rightarrow$气体（降温高压）$\rightarrow$液体（降温）$\rightarrow$固体

例：

$Cl_2$，$N_2$，$NH_3$降温加压$\rightarrow$液氯，液氮，液氨（储存于钢瓶中）

二、分散系

1.分散系

(1):定义：由一种或多种物质分散在另一种物质中所形成的混合系

(2):分散系由分散质和分散剂组成

(3):分类：按分散质粒子直径大小

2.胶体：

(1):定义：分散质粒子直径介于1-100nm的分散系

(2):制备：将饱和$FeCl_3$溶液滴入沸腾的蒸馏水中

$FeCl_3 + 3H_2O(沸水) \stackrel{\Delta}{==} Fe(OH)_3(胶体) + 3HCl$

成功关键：用蒸馏水、适度加热、适度搅拌

(3).特征：

能否透过滤纸 能否透过半透膜 能否发生丁达尔现象 有无吸附性
溶液 能 能 不能 不能
胶体 能 不能 能 能
浊液 不能 不能 不能 不能

注：胶体有吸附性，能吸附其它离子

①丁达尔现象：当光通过分散系时，由于分散质粒子对光散射而在侧面观察到明亮的光线轨迹的现象

②电泳现象：胶体粒子带电，电场作用下，向着与其电性相反的电极移动

③胶体具有一定的稳定性：同种胶体粒子带相同电荷互相排斥

④铁盐，铝盐因水解产生$Fe(OH)_3,Al(OH)_3$胶体，可用于净水

(4):聚沉

①定义：胶体粒子$\rightarrow$较大颗粒物，沉淀析出

②方式：

化学学课的一般思维：类别$\rightarrow$结构$\rightarrow$性质$\rightarrow$用途

1.2 物质的量

一、物质的量

1.物质的量(n)：表示含有一定数目微粒的集合体

符号：n,单位:摩尔(mol)，简称摩

2.阿伏伽德罗常数($N_A$)：12g $\stackrel{}{^{12}C}$ 里所含碳原子个数，12g $\stackrel{}{^{12}C}$里碳原子微粒的集合体所含碳原子数为1mol

符号：$N_A$，单位：$mol^{-1}$

3.微粒数(N): $N = n * N_A$，$n = \frac{N}{N_A}$

二、摩尔质量

1.摩尔质量：1mol微粒所具有的质量

符号：M，单位：g/mol

在数值上与微粒的式量相等

2.质量(m):$m = n * M$，$n = \frac{m}{M}$

三、气体摩尔体积

1.定义：1mol气体所占的体积

符号：$V_m$,单位：L/mol

标准状况($0\stackrel{}{^{o}C}，1.01*10^5$)下，$V_m = 22.4L/mol$

四、影响物质体积大小的因素

微粒数目：S，L，g

微粒大小：S，L

微粒间的距离：g

固、液态物质的体积主要取决于微粒数目和微粒大小

1mol固、液态物质的体积主要取决于微粒大小

气态物质的体积主要取决于微粒数目和微粒间的距离

1mol气态物质的体积主要取决于微粒间的距离

压强增大，气体分子间的平均距离减小，体积减小

五、阿伏伽德罗定律

1.定律：同温同压下，同体积的任何气体都含相同分子数（同物质量）

理想气体状态方程:$P$ $V_气$ = $nRT$

$微粒数(N) \stackrel{N_A}{\rightleftharpoons}物质的量(n)\stackrel{M}{\rightleftharpoons}质量(m)$

$气体体积(V)\stackrel{V_m}{\rightleftharpoons}物质的量(n)$

$n = \frac{N}{N_A} = \frac{m}{M} = \frac{V_气}{V_m}$

2.推论1：同温同压下，$\frac{V_{气1}}{V_{气2}} = \frac{N_1}{N_2} = \frac{n_1}{n_2}$

3.推论2：同温同压下，$D(相对浓度) = \frac{\rho_1}{\rho_2} = \frac{M_1}{M_2}$

六、物质的量浓度

1.定义：以单位体积溶液里所含溶质B的物质的量来表示溶液组成的物理量，叫做溶质B的物质的量浓度

符号：$C_{溶质B},单位：mol/L$

2.表达式：$C_{溶质B} = \frac{n_{溶质B}}{V_{溶液B}}$

注：$n \rightarrow 非溶质质量，V \rightarrow 非溶剂体积$

3.$C_浓$ $V_浓$ = $C_稀$ $V_稀$ ， $C$ = $\frac{1000\rho a \%}{M}$

实验：配置溶液

(1).算:

①配制100mL $1mol/L$ $NaCl$ 溶液

n = C V，m = n M

$m_{NaCl}$ = 1mol/L 0.1L 58.5g/mol = 5.850g

②配制480mL 0,1mol/L $CuSO_4$溶液

$m_{CuSO_4}$ = 0.1mol/L 0.5L 160g/mol = 8g

$m_{五水合硫酸铜}$ = 0.1mol/L 0.5L 250g/mol = 12.5g

③用$18.4mol/L$的硫酸配制500mL $0.1mol/L$的稀硫酸

$C_浓$ $V_浓$ = $C_稀$ $V_稀$，$V_浓$ = $\frac{0.1mol/L*0.5L}{18.4mol/L} = 2.72mL$

选用5mL量筒量取

(2).称固体——电子天平，量液体——XmL量筒

(3).溶解/稀释 ， 冷却——烧杯，玻璃棒

浓硫酸稀释，将浓硫酸倒入装有水的烧杯中，并不断用玻璃棒搅拌

(4).转移——XXmL容量瓶

(5).洗涤——2-3次，倒入容量瓶

(6).定容——1-2cm处改用胶头滴管滴加

误差分析：

操作 n溶质 V溶液 C(浓度)
称取世间过长 变小 不变 变小
未冷却至室温而注入定容 不变 变小 变大
转移时少量溶液流出 变小 不变 变小
烧杯或玻璃棒未洗涤 变小 不变 变小
定容时加水过量用滴管吸出 变小 不变 变小
定容摇匀后液面下降又加水 不变 变大 变小
定容摇匀后，液面下降 不变 不变 不变
实验前，容量瓶内残留蒸馏水 不变 不变 不变

1.3 化学中常用的实验方法

一、物质的制备

1.气体的发生装置


(1).C装置与D相比有何优点？

答：C装置能够通过分液漏斗控制滴液速率，从而控制反应速率快慢，得到均匀平稳的气流

(2).D装置与C相比有何优点？

答：装置D适用于块状固体和液体，不加热的反应，可以做到随开随用，随关随停

2、气体的收集装置

多功能瓶的作用：

(1).能够用排空气法收集气体：密度比空气中的—-长进短出，密度比空气轻的—-短进长出

(2).能用排水法收集气体：装满水，短进长出

(3).能进行气体的净化和干燥：装相应的除杂试剂，长进短出

3.二氧化碳的制备

反应原理：$CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$

思考1：大理石是否可以改成碳酸钠？

答：否，因为碳酸钠粉末状，反应速率过快，不利于形成稳定匀速的气流

思考2：稀盐酸可否用稀硫酸替代？

答：否，硫酸和大理石形成硫酸钙微溶，会附着在大理石表面，阻止后续反应

思考3：若一定要用碳酸钠和稀硫酸制取二氧化碳？对装置应该做何改进？

答：可以用分液漏斗代替长颈漏斗，通过控制盐酸的下滴速度，控制反应速率

思考4：通过大理石和盐酸制取的二氧化碳有杂质吗？是什么？如何除杂？

答：有$HCl$杂质，可以用饱和$NaHCO_3$进行洗气

4.氯气的制备
信息：氯气是一种黄绿色的气体；有毒，能溶于水，但在饱和食盐 水中的溶解度却很小。
反应原理：$MnO_2 + 4HCl(浓) \stackrel{\Delta}{==} MnCl_2 + Cl_2\uparrow + 2H_2O$

思考1：制取氯气时可能需要关注哪些问题？

答：氯气有毒，尾气要吸收，通风厨中进行；浓盐酸易挥发，可能会形成杂质

思考2：浓盐酸易挥发，实验过程中为了液体能够顺利下滴，分液漏斗通常会打开盖子。如何改进才能既保证液体的顺利下滴，又防止浓盐酸的挥发？

答：使用恒压分液漏斗

思考3：如何制取一瓶干燥的，纯净的氯气？

二、 物质的分离和提纯

1.区别:

分离：利用物质的性质差异将其组分彼此分开，得到纯净物组分的过程。

提纯：将混合物中的主要成分（或某种指定物质）净化,而把其它杂质除去。

2.原则：

①不增————提纯过程中不增加新的杂质

②不减————不减少欲被提纯的物质

③易分离————被提纯物与杂质容易分离

④易复原————被提纯物质要复原

3.方法:

①固——固混合分离型：热分解、升华、结晶（重结晶）

②固——液混合物分离型：过滤、盐析、蒸发

③液——液混合物分离型：萃取、分液、蒸馏、渗析

④气——气混合分离型：洗气

4.过滤：

①适用范围：固、液分离

例：除去粗盐水中的泥沙

②实验仪器与用品：漏斗，滤纸，烧杯，玻璃棒，铁架台

③操作要点：一贴，二低，三靠

5.沉淀的洗涤：向过滤器中加蒸馏水至浸没沉淀，待水自动流干，重复2~3次。

6.判断沉淀已洗净的方法：取最后一次洗涤液少许于试管中，加沉淀剂，若无沉淀产生，说明已洗净。

7.判断某离子已沉淀完全的方法：静置后，向上层清液中继续加沉淀剂，若无沉淀产生，说明该离子已沉淀完全。

8.蒸发结晶

①适用范围：溶解度随温度改变而变化不大的物质。如氯化钠

②实验仪器：蒸发皿，酒精灯，玻璃棒，铁架台（铁圈）

③操作要点：

a.蒸发过程中要不断用玻璃棒搅拌

b.蒸发皿内溶液不能超过其容量的2/3

c.出现较多固体时，立即停止加热，用余热蒸干固体。

9.冷却结晶

适用范围：主要用于溶解度随温度下降而明显减小的物质。如硝酸钾，结晶水化合物

思考1：提纯混有少量$KNO_3$的$KCl$

答：取样、加水溶解、蒸发结晶，趁热过滤，洗涤，干燥

思考2：提纯混有少量$KCl$的$KNO_3$

答：取样、加水溶解、蒸发浓缩，冷却结晶，过滤，洗涤，干燥

10.蒸馏

①适用范围：分离互溶、沸点差异较大的液体混合物。例如：石油的分馏，蒸馏含有铁离子的水提纯其中水份

②实验仪器：酒精灯,蒸馏烧瓶，石棉网，温度计，冷凝管，牛角管, 锥形瓶, 铁架台（铁圈、铁夹）等

③注意事项：

（1）加沸石，防止暴沸

（2）温度计水银球插在蒸馏烧瓶支管口

（3）冷凝水的流向：下进上出

11.萃取

①原理：

萃取：利用某溶质在互不相溶的溶剂中的溶解度不同，用一种溶剂把溶质从它与另一种溶剂组成的溶液中提取出来。

分液：把两种互不相溶的液体分开

②仪器：分液漏斗，铁架台（铁圈），烧杯

③萃取剂要求：

（1）两种溶剂要互不相溶

（2）溶质在萃取剂中的溶解度比在原溶剂中的溶解度大得多

（3）溶质、溶剂之间不能反应。

④常用四氯化碳，苯，酒精，汽油作萃取剂

⑤不用酒精作萃取剂，是因为酒精与水互溶

⑥使用分液漏斗前要检验是否漏水

⑦分液时打开分液漏斗的瓶塞，保持漏斗内与大气压一致

⑧下层溶液下口出，上层溶液上口出

补充

1.$Cl_2，Br_2，I_2$难溶于水易溶于有机溶剂

2.酒精与水互溶，另外三个与水不互溶

3.$\rho_汽、\rho_苯 < \rho_水 ， \rho_{CCl_4} > \rho_水$

4.$I_2 \rightarrow 汽油、苯、四氯化碳 \rightarrow 紫红色$

5.$Br_2 \rightarrow 汽油、苯、四氯化碳 \rightarrow 稀：橙红色 ， 浓：红棕色$
